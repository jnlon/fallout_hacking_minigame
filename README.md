Emulates the Fallout series terminal "hacking" minigame.

See [this](https://www.reddit.com/r/dailyprogrammer/comments/3qjnil/20151028_challenge_238_intermediate_fallout/) /r/dailyprogrammer thread

## Usage

cd /path/to/fallout_hacking

lein run
